# SalaryCalculator – расчёт зарплаты преподавателя

## 1. Функциональность
- Выбор должности:
  - Ассистент – 150 руб/ч
  - Доцент – 250 руб/ч
  - Профессор – 350 руб/ч
- Ввод количества отработанных часов.
- Возможность включить/отключить удержание налога 13%.
- Отображение начисленной суммы и суммы налога.
